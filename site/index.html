辅助边界

上下文提取

优先使用 Accessibility 获取聊天上下文，不足时回退到截图 OCR。

本地兜底

LLM 配置缺失或网络失败时，仍提供可用的本地候选回复。

安装提示

下载后在 Android 设备上打开 APK。若系统提示风险或未知来源，请确认 APK 来自本项目 GitHub Releases 后再继续安装。

Release 包不包含任何 API Key。使用 LLM 能力前，需要在应用内配置兼容 OpenAI 的 Base URL、API Key 和模型名。
