# AiReplayMate

AiReplayMate is an Android AI-assisted reply project for the Route B approach.

## Goals
- Read chat context via Accessibility
- Fallback to screenshot + OCR when needed
- Generate AI reply candidates via LLM
- Autofill the selected reply into the input box
- Do not auto-send by default
